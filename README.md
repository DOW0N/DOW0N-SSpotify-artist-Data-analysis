Spotify 아티스트 인기 곡 
분석 데이터 프로젝트


프로젝트 개요

Spotify API와 SQLite 데이터베이스를 활용하여 
특정 아티스트의 인기 곡 데이터를 수집하고 
이를 시각화 하는 시스템을 구축하는 프로젝트입니다. 
사용자는 원하는 아티스트를 입력하면, 
해당 아티스트의 인기 곡을 최대 20곡까지 검색하고, 
이를 SQLite 데이터베이스에 저장한 후, 
그래프를 통해 각 곡의 인기 점수를 시각적으로 확인할 수 있습니다.
